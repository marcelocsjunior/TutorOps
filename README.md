# TutorOps
Descrição:  Sistema automatizado desenvolvido em n8n para apoio ao planejamento e documentação de aulas técnicas em tecnologia da informação, utilizando IA local via Ollama, Google Sheets, Google Drive e Telegram. A solução permite gerar planos de aula, atividades práticas, perguntas de fixação, critérios de avaliação e relatórios
